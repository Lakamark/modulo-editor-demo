# Modulo Editor Playground

This project is a real integration playground for **ModuloEditor**.

It is built with Vite and simulates how the editor would be used in a real application (e.g. Symfony + Vite setup).

## Purpose

* Test the public API of `@lakamark/modulo-editor`
* Validate real-world integration (DOM, CSS, forms)
* Experiment with presets, plugins, and markdown rendering
* Serve as a reference implementation

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Structure

```txt
index.html        # Dev entry (simulates a backend template)
assets/           # CSS / static assets
src/              # Application logic (demos)
```
